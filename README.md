# NoderceWebProject
## This WEB practice project （Vue + Node.JS + Koa + MySQL ）

# [Noderce.com V1.0.0](http://noderce.com)
    目前仅支持 1200px以上分辨率 的屏幕 正常体验，
    目前仅支持 文字阅览
## Start
    MySQL ： UTF-8
    All : npm i
      Vue : npm run dev
      Koa : node server
      
## 使用到的相关技术与框架
    Vue
    Node.js
    Koa
    MySQL

## introduce
    这是一个小型的文章发布项目，采用前后端业务分离的实现方式。版本采取功能式迭代，
    前端的交互方式采用 fetch + FromData， 

## Stateus	
	当前是 v1.x版本，这个版本主要是为了快速上线，所以它的体验会比较糟糕，
	
## Config 
    Vue : src/main.js
    Koa : config.js
	
## Start 
    MySQL ： 编码 UTF-8  排序规则  General_ci ， 建议版本5.5以上
    All : npm i
      Vue : npm run dev  
      Koa : node server

## Technology stack
    Vue 2  ：基于 Vue-Cli 构建
    Node.js 
    Koa2 
    MySQL
